# oki-ferry 🚢

隠岐の島々を愛するすべてのターミナルユーザーへ。
`sl` コマンドのように、隠岐汽船のフェリーがターミナルを颯爽と（あるいは大袈裟に）横切るコマンドです。

![oki-ferry demo image](https://via.placeholder.com/800x400.png?text=Oki-Ferry+Animation+Demo)

## 特徴

- **フェリーしらしま (鬼太郎ver.)**: 精密なASCIIアートで再現された隠岐汽船のフラッグシップ。
- **隠岐の情景**: 随伴するカモメ、遠景に見える隠岐の島々、日本海の波。
- **リッチな演出**: 煙突からのダイナミックな煙、重厚なエンジン振動（スクリーンシェイク）。

## インストール方法 (Homebrew)

現在、独自の Tap を介して提供されています：

```bash
brew tap izumi-h/oki-ferry
brew install oki-ferry
```

## 使い方

ターミナルで以下のコマンドを入力してください：

```bash
oki-ferry
```

## 隠岐について

隠岐諸島（島根県）は、4つの有人島と180あまりの無人島からなる、ユネスコ世界ジオパークに認定された美しい島々です。
このコマンドを通じて、少しでも隠岐の風を感じていただければ幸いです。
